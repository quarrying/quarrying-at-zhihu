利用 Python SymPy 的 `plot_implicit` 可以绘制不等式的图形. 

题目中不等式的图形可以用如下代码绘制:
```python
import sympy
x, y = sympy.symbols('x y')
sympy.plot_implicit(sympy.Abs(x + y) > 2*x + 3*y, (x, -5, 5), (y, -5, 5))
```
